PyPlayer by ripdog

A simple music player with a directory scanner, directory watching, XSPF playlist support. Written in python2.6, with SQLite backend, GStreamer playback core and (someday soon) a ncurses interface. (Gotta settle for raw_input() for now ;) ) If this sounds tempting, don't bother. I'm a newbie developer who writes code bad enough to offend the calmest of python gurus, and this code has hardcoded paths galore. If you enjoy manually installing packages and modifying source files to get a substandard music player to work, be my guest, however.

Dependencies

SQLAlchemy Python 2.6 Gstreamer + Python bindings

Install

Install the necessary packages. Modify the files to point paths to your machine. Run scanner.py and then player.py. Better install procedure coming soon!
